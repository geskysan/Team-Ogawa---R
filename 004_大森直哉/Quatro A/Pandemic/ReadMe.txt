
ゲームタイトル：	Pandemic
ジャンル：      	２Ｄドットイートシューティングゲーム
使用デバイス：  	キーボード
使用言語 :		C
制作環境 : 		Windows10( OS )
			adobe photoshop CC 2015 ( 2D描画制作ソフト )
			Microsoft Visual Stuio 2015 ( プログラミングソフト )
			Microsoft DirectX SDK(June 2010)
制作期間 :		2017年11月 〜  2018年4月 (5ヵ月)

====遊び方====
黄色のプレイヤーを十字キーで移動させて道中に落ちている玉をすべて取ればクリアです
敵が追尾してくるのでうまくよけながら玉を取ってください
玉を10個とると弾をスペースキーで打てるようになります
当たった敵は３秒間動きが止まります

====実行方法====
そのままexeを起動してください

====操作方法====
ENTER					… 決定
SPACE	 	 			… 弾の発射
アローキー				… 前後左右移動


====頑張ったポイント====
Excelでマップを作った点です。0であれば何も置かず
6であれば壁を配置するなどそれぞれに割り当てるのを頑張りました。
